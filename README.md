# Tetris - Rust

## Preview
![Game Image](https://i.imgur.com/1MWAXAK.png)

## Instructions
Execute `cargo build && cargo run` to install the dependencies/build the program and then run it.

The keybindings are `UP, DOWN, LEFT, RIGHT, SPACE, Z, and X`.
